# Screens (fields to show)

## Account 360

* **Header**: legal_name, sap_customer_no, segment, territory, owner
* **Tabs**

  * **Overview**: credit widget (limit, exposure, available, aging), payment terms, negotiated_pricing_flag
  * **Orders**: list with date, value, status; drill to lines
  * **Deliveries/POD**: list + drill
  * **Invoices/AR**: list + aging chart + invoice PDF link
  * **Contacts**: grid (role, phone, email)
  * **Activities**: timeline; add new
  * **Files**: uploaded docs (CRM) and links to SAP DMS (read-only)
